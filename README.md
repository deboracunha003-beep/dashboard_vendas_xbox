# dashboard_vendas_xbox
Este projeto apresenta um dashboard interativo desenvolvido no Excel, com foco no monitoramento das vendas de assinaturas e produtos do ecossistema Xbox. O painel permite uma visualização clara do faturamento e do comportamento dos usuários em relação aos diferentes planos oferecidos.

O arquivo dashboard_vendas_xbox.xlsx está organizado em abas de base de dados, cálculos e dashboard, utilizando tabelas dinâmicas para o processamento das informações e apresentação dos resultados.

Os principais indicadores analisados incluem o faturamento do EA Play Season Pass (R$ 600,00), do Minecraft Season Pass (R$ 940,00) e o comparativo de faturamento do Xbox Game Pass entre assinantes com renovação automática (R$ 1.537,00) e sem renovação (R$ 217,00).
